Guessing Game

About this project

This is the first project that i made to practice my python skills. It is a very simple number guessing game, where the user have to guess a number from 1-100 with 10 trys. This game loops forever until the user exit the game through input.

Features:

  • Randomizing the number to guess for each new game session.
  • Keeping track of the number of guesses made by the user.
  • Providing a hint every guess by telling the user if it is higher or lower.

Mean, Median, Mode

About this project

This project is a simple program that calculates the mean, median, mode from an array of numbers inputed by the user. This mainly was created using for-loop and multidimensional array.

Features:

  • It allows users to input a set of numbers
  • Calculates the mean(average)
  • Calculates the median(middle)
  • Calculates the mode(duplicate)
  • Display the output after performing the calculations
